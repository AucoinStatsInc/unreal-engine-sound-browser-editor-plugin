# Contributing

Thanks for taking the time to help improve **Sound Browser for Freesound.org**!

This GitHub repository is used for **documentation and issue tracking**.  
The plugin itself is proprietary and distributed through the Epic Games Fab Marketplace.

## Pull Requests

This project does **not** accept pull requests.  
If you have a fix or improvement idea, please open an issue describing the change and we’ll consider it.

## Before creating an issue

Please check:
- The existing issues (open and closed)
- The README / FAQ
- That you are running a supported Unreal Engine version

## Bug reports

When reporting a bug, please include as much of the following as possible:

- **Unreal Engine version** (e.g., 5.6.1, 5.7.0)
- **Operating system** (Windows/macOS/Linux)
- **Plugin version**
- **Steps to reproduce** (numbered steps are best)
- **Expected vs actual behavior**
- **Relevant logs**
  - Include `Saved/Logs` excerpts
  - If possible, filter for the plugin’s log category (use `log LogSoundBrowser Log` to enable)
- **Screenshots or screen recordings** (if the issue is UI-related)
- Whether the issue persists after restarting the editor

If the issue involves a specific Freesound.org sound, include:
- The **sound ID** (preferred) or a link to the sound page

## Feature requests

Feature requests are welcome! Please include:
- The problem you’re trying to solve
- Your intended workflow
- Why the current behavior is insufficient
- Any UI/UX suggestions or reference examples

## Security & sensitive information

Please do **not** post:
- Private keys or secrets
- Full API keys (redact them)
- Private project files

If you believe you’ve found a security-related issue, open an issue with a minimal description and we’ll provide a private way to share details.

## Support

For questions and usage help, please prefer:
- The Epic Developer Community forum thread (linked from the Fab listing)
- GitHub issues for reproducible bugs and feature requests
